# 微信公众号 Markdown → HTML 转换器

把 Markdown 文章一键转成公众号后台能直接粘贴的格式。不用再手调样式，不用再担心手机端乱掉。

---

## 这东西解决什么问题

在公众号后台写文章，你遇到过这些事：

- 复制粘贴进去，**字体大小变了**
- 代码块换个行就**全粘在一起**
- 引用块在电脑上看好好的，**手机端没了背景色**
- 加个底色、加个边框，**手机端全丢**

原因是：微信会把 `<style>` 标签和 CSS class 全过滤掉，只认写在元素身上的 `style=""` 内联样式。

这个工具做的事：**把你写的 Markdown → 自动转成"所有样式都写在元素身上"的 HTML。** 复制粘贴到公众号后台，电脑端和手机端效果一致。

## 你需要什么

- 一台 Windows / Mac 电脑
- 安装了 Python 3（没有的话往下看，有安装教程）

## 第一步：安装 Python（如果已经有了，跳过）

### Windows 用户

1. 打开浏览器，访问 https://www.python.org/downloads/
2. 点击黄色大按钮 **Download Python 3.x.x**
3. 下载完成后，双击安装包
4. **重要**：安装界面第一页，**勾选底部的 "Add Python to PATH"**，然后点 Install Now
5. 安装完成后，打开"命令提示符"（按键盘 `Win + R`，输入 `cmd`，回车）
6. 输入 `python --version`，回车。如果出现 `Python 3.x.x` 说明安装成功

### Mac 用户

1. 打开"终端"（在启动台搜索"终端"）
2. 输入 `python3 --version`，回车
3. 如果没装，会弹出安装引导，按提示操作即可

## 第三步：写文章

用任何你喜欢的文本编辑器（记事本、Typora、VS Code、Obsidian 都行），按 Markdown 格式写文章。

### 基础写法

```markdown
# 这是文章标题

这是正文段落。直接写就行，不用管格式。

## 小标题

这样写就是二级标题。

### 加粗

用两个星号把文字包起来，**这段文字会加粗**。

### 引用

> 这是一段引用文字。
> 引用可以有多行。

### 分隔线

三个减号就是一条分隔线：

---

### 表格

| 列1 | 列2 | 列3 |
|-----|-----|-----|
| 数据 | 数据 | 数据 |
```

### 特殊区块（重点）

这个工具能识别三种特殊内容，自动套上好用的样式：

#### 1. 提示词块

把你的 AI 提示词写成多行引用（每行开头一个 `>`），**内容超过 3 行**就会自动变成灰底代码块样式：

```markdown
> 请帮我做以下事情：
>
> 第一步：分析这段文字
> 第二步：提取关键信息
> 第三步：按格式输出
>
> 不要编造内容。
```

#### 2. 行动块

以"**现在打开**"开头的段落，会自动变成蓝底醒目样式（适合放在文章末尾，引导读者做一个具体动作）：

```markdown
现在打开你的简历，找一句你最不满意的经历。

把里面的"负责""参与""协助"圈出来。然后问自己三个问题：我具体做了什么？怎么做的？结果是什么？

回答完这三个问题，你再把那段经历发给AI。你会看到差别的。
```

#### 3. 互动钩子

**整行加粗**的问句（以 `**` 开头和结尾，里面带 `？`），会自动变成居中灰底样式：

```markdown
**你简历里最常写的是"负责、参与、协助"里的哪个？**
```

## 第四步：转换

把写好的 Markdown 文件（比如 `我的文章.md`）放到跟 `wechat_md_to_html.py` 同一个文件夹里。

### Windows 用户操作

1. 打开文件夹，在地址栏输入 `cmd`，回车
2. 在弹出的黑色窗口里输入：

```bash
python wechat_md_to_html.py 我的文章.md 我的文章_发布版.html
```

3. 回车。看到 `✓ 转换完成` 就说明成功了。

### Mac 用户操作

1. 打开"终端"
2. `cd` 到工具文件夹（比如 `cd ~/Desktop/wechat-tool`）
3. 输入：

```bash
python3 wechat_md_to_html.py 我的文章.md 我的文章_发布版.html
```

## 第五步：发布

1. 用记事本（或任何文本编辑器）打开生成的 `我的文章_发布版.html`
2. **全选**（`Ctrl+A`）→ **复制**（`Ctrl+C`）
3. 打开微信公众号后台 → 新建图文 → 在正文编辑区 **粘贴**（`Ctrl+V`）
4. 上传封面图、设置摘要
5. 点"预览" → 发到手机上看效果
6. 确认没问题 → **保存草稿** → 定时发布

## 常见问题

### Q：转换后手机上看，某段文字没有背景色？

A：检查一下那段文字在 Markdown 里是不是以"现在打开"开头（行动块）、或者是整行加粗的问句（互动钩子）。如果不是这些特殊情况，就不会有特殊样式。普通文字就是白底黑字，这是正常的。

### Q：提示词块没有变成灰底？

A：需要满足两个条件：
1. 每行都以 `> ` 开头
2. **总共超过 3 行**（3行或以下会按普通引用处理）

### Q：表格显示不正常？

A：确保 Markdown 里的表格格式正确：
```markdown
| 列1 | 列2 |
|-----|-----|
| 数据 | 数据 |
```
第二行的 `|-----|-----|` 不能省略。

### Q：我想自己调样式？

A：打开 `wechat_md_to_html.py`，找到 `STYLES = {...}` 那一段，修改对应颜色和尺寸，保存后重新运行即可。
